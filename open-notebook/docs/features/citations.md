# Citations & Grounding

The primary value of Open Notebook is **Grounding**. Unlike ChatGPT, which answers from its general training data, Open Notebook answers from _your_ data.

## I. The Retrieval mechanism (RAG)

When you ask a question:

1.  **Search**: The system searches your notebook for relevant text chunks (Vector Search + Keyword Search).
2.  **Context**: It pastes those chunks into the AI's "Context Window."
3.  **Instruction**: It tells the AI: _"Answer the user's question using ONLY the provided context."_

## II. Reading Citations

Every claim in a generated answer includes a citation marker (e.g., `[1]`).

- **Hover**: Hover over the number to see a preview of the source text.
- **Click**: Click the number to open the full source document and scroll to the exact paragraph.

## III. Hallucination Control

If the answer cannot be found in your sources, the AI is instructed to say: _"I cannot answer this based on the provided documents."_

This is a feature, not a bug. It prevents the AI from making things up.
