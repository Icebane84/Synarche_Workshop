# Interface Overview

Open Notebook is designed with a three-pillar layout to balance navigation, work, and context.

## I. The Sidebar (Navigation)

Located on the far left.

- **Notebook Selector**: Switch between different active research projects.
- **Tools**: Access global tools like Settings, Profile, or Admin panels.
- **Theme Toggle**: Switch between Light and Dark mode.

## II. The Main Stage (Workspace)

The center area is where your primary work happens. It changes based on the selected **View Mode**:

### [2.1] 🏠 Home / Dashboard

- Recent activity.
- Quick stats about your knowledge base.

### [2.2] 💬 Chat

- The conversational interface.
- Chat history list.
- Input box for asking questions to your notebook.

### [2.3] 📝 Notes

- List of generated notes.
- Markdown editor for reading and refining notes.

### [2.4] 📚 Sources

- Grid or list view of all documents in the notebook.
- Status indicators (Indexed, Processing, Error).

## III. The Context Panel (The Brain)

(Often located on the right or accessible via drawer)

- Shows metadata about the current active source.
- Displays raw text chunks during retrieval debugging.
- Allows you to see exactly _what_ the AI is reading.
