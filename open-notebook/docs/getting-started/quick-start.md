# Quick Start Guide

Get up and running with Open Notebook in under 5 minutes.

## I. Prerequisites

- **Windows 10/11** (for this guide)
- **Python 3.10+** installed
- **Git** installed

## II. The Automatic Way (Windows)

We have created an automated script to handle everything for you.

### [2.1] Execution Steps

1.  **Open PowerShell** and navigate to the project folder.
2.  **Run the startup script**:
    ```powershell
    .\dev_start.ps1
    ```

### [2.2] System Response

The script triggers the following sequence:

- **Detection**: Checks for your Python virtual environment (`.venv`).
- **Healing**: Installs any missing dependencies automatically.
- **Launch**: Spawns 4 separate windows:
  1.  **SurrealDB**: The database engine.
  2.  **API Server**: The backend logic.
  3.  **Worker**: Background task processor.
  4.  **Streamlit App**: The user interface.

### [2.3] Verification

1.  **Open your browser**:
    Go to `http://localhost:8501`.

## III. Success State

You should see the Open Notebook welcome screen.
