# Sources & Knowledge

Open Notebook is "Source-Grounded," which means it relies on the data you feed it.

## I. Supported Formats

| Format       | Description                                                                  |
| :----------- | :--------------------------------------------------------------------------- |
| **PDF**      | Research papers, ebooks, reports. Text is extracted via OCR/Layout analysis. |
| **Web Link** | Articles, blog posts. Content is scraped and cleaned.                        |
| **YouTube**  | Video URLs. Transcripts are automatically downloaded.                        |
| **Text/MD**  | Raw text files or markdown notes.                                            |

## II. Adding a Source

1.  Navigate to the **Sources** tab.
2.  Click **Add Source**.
3.  Choose the method (Upload or Link).

### [2.1] Processing Status

When a source is added, it goes through a pipeline:

1.  **Pending**: Added to queue.
2.  **Processing**: Extracting text and generating embeddings.
3.  **Indexed**: Ready for search and chat.
4.  **Failed**: Something went wrong (check logs).

## III. Managing Sources

- **View**: Click a source card to see its extracted text.
- **Delete**: Remove a source. It will no longer be used for answers.
- **Edit Metadata**: Update the title or author (useful for citations).
