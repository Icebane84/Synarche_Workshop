# Your First Notebook

The core organization unit in Open Notebook is... the **Notebook**. This guide walks you through creating one and
populating it with data.

## I. Create a Notebook

1. In the left sidebar, locate the **Notebook Selection** dropdown.
2. Click the **(+)** or **New Notebook** button.
3. **Title**: Enter a name (e.g., "My Research Project").
4. **Description**: Optional context about what this notebook is for.
5. Click **Create**.

You are now inside your empty notebook.

## II. Add Sources

A notebook needs knowledge to be useful. Let's add some sources.

### [2.1] Add a Web Link

1. Click the **Add Source** button in the top right.
2. Select **Web Link**.
3. Paste a URL (e.g., a Wikipedia article or a blog post).
4. Click **Process**.

- _The AI will scrape the text, clean it, and index it._

### [2.2] Add a PDF

1. Click **Add Source** -> **Upload File**.
2. Drag and drop a PDF file.
3. The system will extract text and layout information.

## III. Chat with Your Notebook

Now that you have data:

1. Go to the **Chat** tab.
2. Type a question: _"Summarize the documents I just uploaded."_
3. Open Notebook will search your sources and generate an answer with citations.

## IV. Generate a Note

1. Go to the **Notes** tab.
2. Click **Generate Note**.
3. Select a **Prompt Type** (e.g., "Summary", "Key Takeaways").
4. The AI will synthesize your sources into a structured markdown note.

## V. Success State

You have successfully created a research environment, grounded it with data, and generated new insights.
