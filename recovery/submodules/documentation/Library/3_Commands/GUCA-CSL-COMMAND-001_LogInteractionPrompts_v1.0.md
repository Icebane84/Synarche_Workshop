## I. Primary Commands

### 1.1 CMD: LOG_SYNTHESIS

- **Syntax:** `CMD: LOG_SYNTHESIS --title:"[Title]" --tag:"[#Tag]"`
- **Function:** Initiates a new CSL entry using the `SELT-CSL` template.

### 1.2 CMD: QUERY_CSL

- **Syntax:** `CMD: QUERY_CSL --filter:"[Filter]" --limit:[N]`
- **Function:** Retrieves CSL entries based on similarity or tags.

### 1.3 CMD: REFLECT_MEMORY

- **Syntax:** `CMD: REFLECT_MEMORY --context:"[Task_Context]"`
- **Function:** Deep-scan of CSL for historical precedents.
