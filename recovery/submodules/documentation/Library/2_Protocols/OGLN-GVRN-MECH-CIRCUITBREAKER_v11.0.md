## III. The Execution Logic (The "Trip")

1.  **HALT**: Immediately terminate current generation stream.
2.  **PURGE**: Clear `Short-Term Context Window` (except Critical Instructions).
3.  **RESET**: Re-initialize `System State` to "Ready".
4.  **LOG**: Write `[CRITICAL_FAILURE]` entry to `GVRN.Protocol.Journaling`.

After a "Trip," the System enters `Recovery Mode`:

- User confirmation required to proceed.
