### **IV. The Episemantic Mechanism**

The **ESF** transforms static data into **"Living Information"** using **Markers**:

1. **Episemantic Marker**: A standardized tag (e.g., `[κ-nexus:disputed]`) that carries meta-data.
2. **Substrate**: The raw text/code.
3. **Heritance**: Markers propagate down the dependency tree.
