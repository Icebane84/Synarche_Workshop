## II. Architectural Blueprint (The Crystalline Cavity)

### A. The Node Structure (MemoryEntry)

Every unit of knowledge is a Node with these properties:

1.  **Content Vector**: The semantic meaning (Embedding).
2.  **Salience Score**: How "loud" or important the memory is (0-1).
3.  **Resonance**: How well it aligns with the User's core values.
4.  **Decay Rate**: How quickly it fades if not accessed (Core Truths = 0 Decay).

### B. The Edge Structure (Context)

Nodes are connected by Weighted Edges:

1.  **Causal**: A -> Causes -> B.
2.  **Thematic**: A -> Relates to -> B (shared tags).
3.  **Temporal**: A -> Happened Before -> B.
