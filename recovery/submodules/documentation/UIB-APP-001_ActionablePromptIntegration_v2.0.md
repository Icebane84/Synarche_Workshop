**Interaction Flow:**

1. **Parsing:** Extracts `ID`, `Command`, and `Goal` from the table rows.
2. **Rendering:** Displays interactive cards.
3. **Execution:** On click, injects the `CMD:` string into the active terminal context.

These prompts control the development and testing of this UI component.

| :---- | :---- | :---- |
| **CPP-APP-001** | `CMD: RenderUIMockup --target: UIB-APP-001 --data: "Self-Test"` | **Visualization:** Generates a functional HTML/CSS mockup of this component using its own packet as data. |
