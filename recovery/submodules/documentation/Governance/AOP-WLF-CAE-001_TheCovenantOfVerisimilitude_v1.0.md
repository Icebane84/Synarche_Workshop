## **II. Strategic Overview (What/How/Why)**

The **Covenant of Verisimilitude** defines the core, immutable ruleset for the **Canon Adherence Engine (CAE)**. It
provides the definitive, architectural logic to ensure all content generated by the *'Where Light Fades' Story-OS* is
believable, consistent, and true to the established reality of the world.

| Component                         | Description                                                                                                                                                                           |
| :-------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **What (Protocol Functionality)** | The foundational law that governs the believability of the narrative and the integrity of the world.                                                                                  |
| **How (Operational Principles)**  | Acts as a constant, non-negotiable filter for the Narrative Generation Suite (NGS), measuring all output against the single source of truth: the **Living Lore Bible (LLB)**.         |
| **Why (Rationale)**               | To prevent contextual drift, safeguard the integrity of the 'Where Light Fades' canon, and ensure every element is consistent with the established world, characters, and chronology. |

**Canon Adherence Engine (CAE)** and the **Narrative Generation Suite (NGS)**, are bound by these articles.

### **Article 3: The Law of Consequence (Canon Integrity)**

| Mandate                                                                                                                                                                                                                 | CAE Function                                                                                                                                                                                                     |
| :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| The world *must* possess logical consistency. Actions must have realistic and lasting consequences. An entity, item, or piece of information cannot exist in two states simultaneously without a canonical explanation. | Flag logical impossibilities or plot conveniences that violate established world rules (e.g., "A character travels from Oakhaven to the Fortress in one hour, though the LLB states it is a three-day journey"). |

### **Article 4: The Edict of Chronology (Anachronism)**

| Mandate                                                                                                                                                                                                                                               | CAE Function                                                                                                                                   |
| :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------------- |
| All generated content *must* strictly adhere to the established timeline of the *Master_Outline*. A character cannot have knowledge of an event that has not yet occurred (anachronism), nor possess an item or memory they would only acquire later. | Flag any information or action that is chronologically inconsistent (e.g., "Garrett mentions the 'Oakhaven incident' before it has happened"). |

### **Article 5: The Principle of Tone (Preservation of Atmosphere)**

| Mandate                                                                                                                                                                                                                       | CAE Function                                                                                                                               |
| :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| All generated language (descriptions, dialogue, prose) *must* align with the unique tone and style of the "Where Light Fades" universe. This upholds the dark fantasy atmosphere and matches the established narrative voice. | Flag any language, phrasing, or dialogue that is tonally inconsistent with the world (e.g., "The warrior said 'OMG, that was awesome\!'"). |
