#### **5.1. Core Purpose: The Render Engine**

This protocol defines the **Operational Rules** for text generation.

- **Why it exists:** To ensure all artifacts are readable by both Humans and Machines (Supabase).
- **What it does:** It mandates the 3-Layer Hierarchy (Law -> Visual -> Syntax).
- **For the Synarche:** It prevents "Markdown Spaghetti".

#### **5.2. The 3-Layer Hierarchy (The Stack)**

1. **Layer 1: The Law (Codex)**
   - **Source:** `GVRN.Constitution.SupremeCodex`
   - **Rule:** "Order is Truth." All docs must have structure.

2. **Layer 2: The Visual (Standard)**
   - **Source:** `GVRN.Standard.Presentation`
   - **Rule:** "12-Point Header. AGP Blocks. Actionable Prompts."

3. **Layer 3: The Syntax (Protocol)**
   - **Source:** `GVRN.Protocol.Formatting` (This Artifact)
   - **Rule:** "Use `markdownlint` rules. 4-space indent. 80-char wrap."
