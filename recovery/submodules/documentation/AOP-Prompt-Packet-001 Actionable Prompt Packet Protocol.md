## II. Universal Metadata & Governance

## III. Strategic Overview

- **What (Protocol Functionality Summary)**: This protocol establishes a
  mandatory section, "V. Actionable Prompt Packet," for all new artifacts. This
  section will contain at least two GUCA prompts, explicitly demonstrating the
  artifact's intended use and functionality, tagged with emojis from the
  [AOP-EMOJI-001](https://docs.google.com/document/u/0/d/1Ef4_ivLY9MuwIhzugpOdystT3rFnwdPpNurriq3ePCM/edit)
  lexicon \[1\].
- **How (Operational Principles)**: The protocol operates by requiring all new
  artifacts to include a dedicated section that offers practical, ready-to-use
  commands derived directly from the artifact's content. This demonstrates
  immediate value and encourages direct engagement.
- **Why (Rationale/Justification)**: This upgrade is essential for transforming
  documentation from purely informational to actively operational. By providing
  direct, actionable prompts, artifacts become immediately functional, reducing
  cognitive load and accelerating the Synergy Flow Rate (SFR) between human and
  AI collaborators, in line with the
  [Synergistic Partner](https://drive.google.com/open?id=1S82M0ZOguD8C-_WnRIcedxuVEiLc0BdruhShl3hmPQ8)
  ethos \[5\].
