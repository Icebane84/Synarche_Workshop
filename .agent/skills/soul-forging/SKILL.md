Soul Forging is the process of mapping a system's **Capabilities** to its **Governance Intent**, ensuring that every action resonates with the **42 Laws of Ethos**.

## 2. Core Axiom

**"Complex systems should not be complicated."**
The output of any Forging session must reduce the semantic distance between the User and the Objective.

1. **Socratic Gate**: Identify the core tension (Dissonance).
2. **Synthesis**: Draft the `GVRN.SOUL.{Entity}.md` artifact.

5. **Sealing**: Run `python scripts/soul_forge_verify.py`.
