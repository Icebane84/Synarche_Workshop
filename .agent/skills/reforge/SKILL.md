# Reforge Skill

This skill allows the agent to standardize artifacts to the Phoenix Protocol v13.0, updating headers, geometry, and synergy blocks.

## Tools

### `reforge_artifact`

Reforges a target file to v13.0 compliance.

- **Parameters**:
  - `target_file` (string): The file to reforge.
  - `domain` (string, optional): The domain code (default: `GVRN`).
  - `type` (string, optional): The artifact type (e.g., `Protocol`, `Blueprint`).

- **Usage**:
  ```python
  reforge_artifact(target_file="docs/MyDraft.md", domain="GVRN", type="Protocol")
  ```
