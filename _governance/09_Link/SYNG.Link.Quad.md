### **[ARTIFACT START]**

By mandating the **Quad-link structure**, the system ensures every connection possesses exhaustive, unambiguous insight into its historical and logical context.

## 🧬 Anatomy of a Link

A Quad-Link consists of four mandatory components:

1. **[Source Node]**: The anchor point where the link originates (typically the active artifact).
2. **[Relationship Enum]**: The specific edge type defined in [SYNG.Link.Semantic](SYNG.Link.Semantic.md).
3. **[Contextual Description]**: A high-fidelity explanation of _why_ the link exists and what it achieves.
4. **[Target Node]**: The destination artifact or entity.

## 🛠️ Implementation Example

```yaml
LINK-ID: GVRN-LINK-772
source: "GVRN.Master.Registry"
type: "GOVERNS"
context: "The Registry provides the single source of truth for all project artifacts."
target: "SYNG.Link.Semantic"
```

## 🚀 Actuation Rule

Every link in the governance substrate MUST be decomposable into a Quad-Link to be eligible for canonization.
