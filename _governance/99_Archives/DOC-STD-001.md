graph.

## **III. The Architectural Formatting Standards**

### **3.1. Markdown Syntax (The 4-Space Mandate)**

All markdown files must utilize **4-space indentation** for nested lists and sub-items. This is a hard requirement for
parser stability.

### **3.2. Section Hierarchy & Bolding**

Section headers must be bolded within the header tag to create visual and structural weight.

- **Header Standard:** `## **X. Section Name**`
- **Sub-Header Standard:** `### **X.Y. Sub-Section Name**`

### **3.3. OGLN Artifact Delimiters**

Every formal artifact must be wrapped in bolded delimiters to clearly define the boundary of the canonized text.
