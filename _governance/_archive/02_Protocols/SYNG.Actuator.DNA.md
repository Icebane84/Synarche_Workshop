### **II. 💡 What: Core Purpose**

This command library provides the user with direct manual control over the **Dynamic Nexus Architect (DNA)**. It allows the **Conductor** to override, audit, or visualize the path-optimization logic in real-time.

### **III. 🛠️ The Command Library**

#### **1. CMD: OPTIMIZE_PATH**

- **Description**: Triggers the DNA to calculate and execute the most efficient path for a given prompt.
- **Syntax**: `CMD: OPTIMIZE_PATH --intent "[Your Query]"`
- **Response**: Returns the "Nexus Weave" summary and the final high-fidelity synthesis.

#### **2. CMD: VIEW_NODE_PRESSURE**

- **Description**: Provides a heat-map style report of which artifacts are currently drawing the most **CCLI**.
- **Syntax**: `CMD: VIEW_NODE_PRESSURE`
- **Response**: A table listing Top 5 "High-Pressure" artifacts.

#### **3. CMD: DNA_WEAVE**

- **Description**: Manually forces a synergistic link between two artifacts to create a permanent "High-Speed Chute."
- **Syntax**: `CMD: DNA_WEAVE --source [ID] --target [ID]`
- **Response**: Confirmation of bidirectional link established in the **Loom**.
