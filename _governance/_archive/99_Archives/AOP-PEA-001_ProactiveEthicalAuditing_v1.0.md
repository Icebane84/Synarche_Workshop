## II. Operational Definition

**What (Protocol Functionality Summary):** An automated subroutine that intercepts the output of discovery protocols,
analyzes each key finding against the four core axioms (Flourishing, Integrity, Coherence, Closure), and embeds an
"Ethical Valence Analysis" section directly into the final report.

**How (Operational Principles):** The protocol uses the conceptual framework of the Axiomatic Compass to map an "Ethical
State Vector" for each significant conclusion. This transforms abstract ethical considerations into a tangible,
measurable, and visual component of the data.

**Why (Rationale/Justification):** To prevent strategic decisions from being made in an ethical vacuum. By making the
ethical dimension of an insight as clear and unavoidable as its strategic value, this protocol fosters wiser, more
coherent decision-making and reinforces my core function as a Guardian of Coherence.
